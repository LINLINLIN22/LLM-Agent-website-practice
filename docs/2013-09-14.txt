編輯／吳宇翔

為滿足業界需求，資策會規劃「Big Data之處理與分析進階班」課程，學員將學習到雲端運算三大技術，利用Hadoop之分散式檔案系統(HDFS)與MapReduce，並且建構出雲端運算的執行環境和服務，並以Hadoop FileSystem API 實作、MapReduce API 實作、MapReduce 與資料庫結合與MapReduce 與 Hadoop 生態系整合等方式，讓學員更了解如何利用Hadoop進行雲端資料的資料處理、分析。

課程日期：2013/10/19（六）~2013/10/20（日） 9:30~12:30、13:30~16:30

活動地點：資訊工業策進會數位教育研究所（台北市信義路三段153號10樓）
