標題: HTC 將於二月推出 NFT 交易市集 VIVE BYTES
原文: https://www.ectimes.org.tw/2022/01/hct-%e5%b0%87%e6%96%bc%e4%ba%8c%e6%9c%88%e6%8e%a8%e5%87%ba-nft-%e4%ba%a4%e6%98%93%e5%b8%82%e9%9b%86-vive-bytes/
宏達國際電子(HTC) 旗下 VR 應用程式商店平台VIVEPORT於 1 月 25 日宣布，其將於 2 月中旬推出 NFT 交易市集 VIVE BYTES。消費者可透過 VIVE BYTES 購買 NFT 數位商品，另外也推出 NFT 盲盒機制，未來還會開放拍賣的功能。根據規劃，VIVE BYTES 市集起步將針對動畫、卡通和遊戲等領域推出相關 NFT 數位收藏。

￭宏達電旗下平台VIVEPORT將推出 NFT 交易市集 VIVE BYTES。（截圖來源 /VIVEPORT 官方網站）

宏達電表示，VIVE BYTES 的推出，將方便全球使用者輕鬆進入 NFT 的世界。VIVE BYTES 提供友善且直覺的購物流程，以及獨特的遊戲化功能，即使是未曾接觸 NFT 交易的消費者，也能如同網購一般輕鬆上手。消費者可以利用法定貨幣和加密貨幣進行支付，同時可串聯MetaMask錢包，為 NFT 商品提高安全性和便利性。

首波 VIVE BYTES 市集交易的NFT 收藏內容為日本知名畫家山本修的 CAT ART 作品，同時《Step into Cat Art 走進喵次元》貓‧美術館展覽已在 VIVEPORT 如火如荼展開，消費者可透過手機、電腦、VR 頭戴裝置等上網參觀。消費者可於欣賞展覽的同時，也能進入 VIVE BYTES 市集購買喜歡的 CAT ART NFT 收藏。

VIVEPORT 總經理林俊吳表示：「NFT 正在改變我們對於身份和所有權的看法，而 VIVE BYTES 正居於第一線，聚集並可激發創作者無限創意數位化的潛力，且使消費者更容易得取得數位資產」。
