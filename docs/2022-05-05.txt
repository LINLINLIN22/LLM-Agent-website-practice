標題: 用 LINE 聊天買快篩　找找快篩試劑幫你找庫存
原文: https://www.ectimes.org.tw/2022/05/%e6%89%be%e6%89%be%e5%bf%ab%e7%af%a9%e8%a9%a6%e5%8a%91-%e5%be%85%e6%9b%b4%e6%96%b0/
因應家用快篩試劑的需求增加，中央於 4 月 28 日宣布快篩試劑販售實名制開始上路。繼口罩實名制販賣之後，民眾到藥局排隊的現象又捲土重來，即使透過實名制的方式限制購買數量，仍然有許多民眾買不到，甚至跑遍各個藥局皆以販售完畢。面對此情況，資深程式設計師陳葵懋，利用Microsoft Azure製作「找找快篩試劑」LINE 聊天機器人，方便民眾透過簡易聊天的方式，找尋附近的藥局及庫存。

￭ 資深程式設計師陳葵懋製作 LINE 聊天機器人方便找尋藥局及庫存 。（截圖來源 / 賴曉芸）

2020 年疫情爆發之時，陳葵懋就已透過「敗口罩」 LINE 聊天機器人的方式，讓民眾便利搜尋附近藥局的口罩庫存。如今，陳葵懋複製「敗口罩」的成功經驗製作出「找找快篩試劑」。陳葵懋表示，選擇開發 LINE 聊天機器人主要考量台灣人的社群使用習慣，不僅操作門檻低，更可以省下設計 UI （User Interface，使用者介面）的成本，並更專注於資料的串聯。

「找找快篩試劑」透過 Microsoft Azure 的 AI 平台進行搭建，同時加入 AI 認知服務的「問題解答 (Question Answering)」。使用者只要在聊天室詢問何處可購買快篩試劑，並按照解答分享定位，系統就會自動顯示附近的藥局及庫存。另外，陳葵懋也加入「自然語言理解 (Language Understanding)」的功能，讓 LINE 機器人快速辨識使用者傳送的文段是屬於抱怨、詢問或地址查詢等，並依照文字提供相對應的內容。

陳葵懋說明，此系統主要串接政府公用資料集，有可能因為大量的使用量，產生延遲或系統當機等問題，而聊天機器人直接連結 Azure 資料庫，可利用現有資料快速回應使用者的詢問。除此之外，系統後台可利用Power BI分析快篩試劑的數據，未來也許可協助資源的配送。
