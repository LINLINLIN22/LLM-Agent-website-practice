標題: Windows  Live  Search     對Google不具威脅?
原文: https://www.ectimes.org.tw/2006/05/windows-live-search-%e5%b0%8dgoogle%e4%b8%8d%e5%85%b7%e5%a8%81%e8%84%85/
記者／陳冠良

你最愛用的搜尋引擎是哪一個？根據iTWire所做的民調顯示，有58.1％的網路使用者不認為微軟在搜尋引勤市場上能打贏Google，而41％的網路使用者認為微軟可能成為Google的威脅。

為了跨足網路搜尋市場，或取消相關的廣告或關鍵字行銷利益，先前微軟已合併了MSN 和Windows 二個部門，並提供Windows Live 的Web服務。而新的Windows Live Search 能夠搜尋圖像、新聞、RSS、電子郵件，並具有本地搜尋功能。用戶可更可以透過Windows Live Messenger、Windows Live Mail 在內的所有Windows Live服務使用Windows Live Search的搜尋功能。

微軟在搜尋市場上的資源投入是有目共睹的，如Windows Live Search的推出、即將上市的新瀏覽器Internet Explore7，以及微軟的強力武器新一代作業系統Vista都將聯結上企業搜尋功能等，在在表明了微軟必勝的雄心與對搜尋市場的重視。

搜尋市場的龍頭老大Google也感受到了微軟的排山倒海攻勢，美國司法部於日前宣佈微軟的瀏覽器Internet Explore7的搜尋服務並無涉及不正當競爭。這場搜尋市場大戰誰能勝出，就看使用者如何回應了。
