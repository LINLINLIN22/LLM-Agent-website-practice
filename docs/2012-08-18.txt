記者／王端玉

三星於16日在台灣發表GALAXY Note 10.1 平板電腦，並將其定位為跨越智慧型手機和平板電腦的新產品線，主要鎖定企業和學生族群。GALAXY Note 10.1提供了手寫數位筆「S Pen」，提供使用者更多編輯和創作的運用，可望開拓平板市場新商機，希望能與蘋果iPad相抗衡。

GALAXY Note 10.1是從智慧型手機GALAXY Note演化而來，並為三星首款搭配手寫數位筆的平板電腦。此款平板搭載Android4.0作業系統，並且設有眼球移動偵測系統，避免在使用中進入休眠模式。此外，支援「多重視窗應用」是一大賣點，可將螢幕一分為二，同時開啟兩種程式，還可再以彈跳視窗的方式，開啟內建的8種Mini Apps。

曾發表多款平板電腦的三星，一直攻不下蘋果iPad佔據的市場。此款GALAXY Note 10.1，以iPad為假想敵，雖然外型差異不大，但新增了許多IPad沒有的功能。根據科技研究調查機構IHSiSuppli的2012年銷售量調查，蘋果iPad的市占率達64%，而三星則佔10%，未來GALAXY Note 10.1想要追上蘋果，還有很大的空間需要努力。

台灣三星行動通訊部總經理杜偉昱表示，三星於2011年推出GALAXY Note，並於全球熱銷，而新上市的 GALAXY Note 10.1 致力以手寫數位筆的優勢，掀起下一波新浪潮，並帶給使用者全新體驗的創意生活。三星攻佔平板市場，是否能拉近與蘋果iPad的距離，還有待市場銷售的考驗。
