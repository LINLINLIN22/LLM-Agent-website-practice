標題: 出師不利　三星note7頻傳爆炸停止出貨
原文: https://www.ectimes.org.tw/2016/09/%e5%87%ba%e5%b8%ab%e4%b8%8d%e5%88%a9%e3%80%80%e4%b8%89%e6%98%9fnote7%e9%a0%bb%e5%82%b3%e7%88%86%e7%82%b8%e5%81%9c%e6%ad%a2%e5%87%ba%e8%b2%a8/
記者／吳宜倫

南韓手機大廠三星（SAMSUNG）八月上市的全新智慧型手機Galaxy Note 7，截至目前為止，已傳出多達35起的爆炸事件，大多是充電時起火所導致的爆炸，三星行動部門總裁高東貞(Koh Donjing)在昨天下午宣布，Note7將暫時全面停止銷售，並在全球地區進行召回，替消費者更換新機。

據三星的聲明表示，目前全球已發生35起Galaxy Note 7疑似爆炸的事件，起因還在詳細調查，但基於多數案件與電池有關，三星也將與電池供應方一同了解原因，基於維護消費者的安全，三星決定全面暫停銷售 Galaxy Note 7，並且開始召回，對於已經入手Note 7 的消費者，未來將可免費更換新機，而台灣用戶的部分，三星預計9月中陸續接受換機的申請。

￭三星Galaxy Note 7手機，目前已傳出多達35起的爆炸事件。（截自／官網）

Galaxy Note 7目前在全球已經出貨約 250 萬支，已交到用戶手上的則約有 100 萬支，儘管根據韓國聯合通訊社（Yonhap News Agency）的報導指出，有問題的 Galaxy Note 7在總出貨量中只暫0.1%，不過為了挽回品牌形象以及重獲消費者信任，三星還是決定召回產品並停售Galaxy Note 7。

此一事件除了衝擊出貨量之外，也可能導致三星股價下滑，且多數人認為三星Galaxy Note 7是三星為了與蘋果公司即將推出的iPhone 7抗衡，所推出的旗艦機，此時爆發電池問題，可說是打折了Note 7原本能為三星帶來的獲利與市場影響力。

不過，三星並非全盤皆輸，消費者鍾沂蒨就表示，對於Note 7這次所推出的虹膜辨識以及防水功能相當有興趣，雖然蘋果公司的iPhone系列評價一直很好，但自己主要還是習慣使用三星系列，因此未來仍不排除購買三星所推出的手機，只是會在手機推出後觀望一陣子才下手，不會成為首購族。

雖然忠實的消費者仍有一定數量，但這樣觀望的行為模式將對出貨量產生首當其衝的影響，三星要如何重獲消費者信任，只能看屆時Note 7後續的品質表現是否能贏回民心。
