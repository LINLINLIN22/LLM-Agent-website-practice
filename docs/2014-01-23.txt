標題: 虹膜辨識夯　蘋果三星致力研發
原文: https://www.ectimes.org.tw/2014/01/%e8%99%b9%e8%86%9c%e8%be%a8%e8%ad%98%e5%a4%af%e3%80%80%e8%98%8b%e6%9e%9c%e4%b8%89%e6%98%9f%e8%87%b4%e5%8a%9b%e7%a0%94%e7%99%bc/
記者／陳思穎

目前許多手機業者致力於更安全的辨識系統，三星Galaxy S5先前運用指紋辨識模組，讓用戶以觸控螢幕的方式進行指紋辨識。蘋果iPhone5s 也採用Touch ID 指紋解鎖方式，然而Touch ID 辨認指紋需要花費幾秒鐘，若是手指潮濕，就會導致辨識失效。

虹膜辨識可以有效改善，Touch ID 辨識時間和辨認精準度的缺陷。MacRumors先前報導，虹膜的結構組成，如同指紋具有獨特性，便於辨別身分。虹膜辨識花費的時間比指紋辨識少、更可以在數呎之外掃描，移動時也可以使用。然而使用虹膜辨識技術不但需要在光線充足之處才能運用，同時引發隱私的爭議。

由於虹膜辨識處理的數據點，比指紋多出20倍之多，因而判定結果較為精準。不僅蘋果積極開發虹膜辨識，三星也和韓國延世大學生物劑量工程研發中心，共同打造「虹膜辨識技術」。三星於去年5月申請虹膜辨識專利，但是否在未來產品搭載指紋辨識技術，三星目前仍無正面回應。

隨著智慧型手機的普及，其安全問題使許多手機廠商投入相關技術研發。三星Galaxy Note 3和Galaxy S4 強打眼球辨識功能，而今年蘋果iPhone 5s和宏達電等品牌也基於安全問題，採用指紋感應系統。關於虹膜辨識，MacRumor 指出，虹膜掃描的運用，可能還要過幾年才能運用於智慧型手機中。
