標題: Yahoo Search Pad反擊　搜尋市場火藥味濃

記者／朱晏瑱隨著微軟的Bing在各國與Google較勁下搶下部份市佔率，Yahoo！也不徨多讓於前日（7日）推出Yahoo Search Pad網路筆記工具，此款新服務的推出不僅暗示了搜尋市場砲火開戰，也嗅出Yahoo！企圖以功能和其它競爭對手來做區分進行反擊。Yahoo Search Pad置於Yahoo！搜尋結果頁的角落，可整理使用者所搜尋的結果並自動分類儲存與記錄，和過去Google Notebook與JetEye需使用者主動設定，較不同的是它能夠智慧化地去辨別與觀察使用者的搜尋行為，再以科學角度去分析和判別使用者搜尋的動作，究竟是屬於純粹打發時間胡亂搜尋，還是針對專門類別題材執行搜尋，再行記錄。Yahoo Search Pad所儲存的筆記內容可包含文字敘述、圖片與相關連結等等，並可自行將此份筆記自由添加、刪除、修改或轉寄印出，甚至與好友分享，即便使用者瀏覽其它網站，依然可運用這份筆記。因此對於使用者再後續欲搜尋相關主題類別時，此份筆記有較大的利用價值，非常適合欲進行複雜主題搜尋的使用者來運用。此項新服務的推出雖然在搜尋市場掀起一番波浪，但因之前Google發表類似於Google Notebook可存取記錄的服務已遭停用。因此Yahoo Search Pad所能達到的成效，與使用者的需求是否精準對味都還有待觀察，業者目前仍採保留態度居多。

記者／朱晏瑱

隨著微軟的Bing在各國與Google較勁下搶下部份市佔率，Yahoo！也不徨多讓於前日（7日）推出Yahoo Search Pad網路筆記工具，此款新服務的推出不僅暗示了搜尋市場砲火開戰，也嗅出Yahoo！企圖以功能和其它競爭對手來做區分進行反擊。

Yahoo Search Pad置於Yahoo！搜尋結果頁的角落，可整理使用者所搜尋的結果並自動分類儲存與記錄，和過去Google Notebook與JetEye需使用者主動設定，較不同的是它能夠智慧化地去辨別與觀察使用者的搜尋行為，再以科學角度去分析和判別使用者搜尋的動作，究竟是屬於純粹打發時間胡亂搜尋，還是針對專門類別題材執行搜尋，再行記錄。

Yahoo Search Pad所儲存的筆記內容可包含文字敘述、圖片與相關連結等等，並可自行將此份筆記自由添加、刪除、修改或轉寄印出，甚至與好友分享，即便使用者瀏覽其它網站，依然可運用這份筆記。因此對於使用者再後續欲搜尋相關主題類別時，此份筆記有較大的利用價值，非常適合欲進行複雜主題搜尋的使用者來運用。

此項新服務的推出雖然在搜尋市場掀起一番波浪，但因之前Google發表類似於Google Notebook可存取記錄的服務已遭停用。因此Yahoo Search Pad所能達到的成效，與使用者的需求是否精準對味都還有待觀察，業者目前仍採保留態度居多。
