記者／黃逸珊

台灣微軟於5日首度針對專案管理工具Office Project推出認證，並和長宏專案管理顧問公司合作，向企業管理人員推廣課程。

近年來台灣考取PMP（Project Management Professional）專案管理師的人數不斷增加，台灣微軟產品行銷經理張立業表示，台灣去年考取PMP的人數成長了49%，在全球排名第十位，顯示台灣對於專案管理人才的需求已邁入國際水平。但是台灣專案管理實務界卻仍然缺少專案管理工具應用認證。

目前台灣已有四千多人取得專案管理師證照，專案管理師證照是一項國際性專案管理方法論的認證。而台灣取得PMP的人數在去年就增加了3347人。微軟看中專案管理在市場上的需求，因此首度推出專案管理工具Office Project認證技術專家證照MCTS。

微軟五月間於全球推出微軟認證技術專家，負責推廣認證的微軟專案經理陳俊志表示，MCTS-Project 2007乃是該公司自有MCTS證照體系下，第一個針對專案管理工具推出的國際證照。 由於這項證照剛推出，目前全球通過認證者約為100人，台灣七月初剛有第一批約30多人取得認證，即由長宏專案管理顧問公司培訓而成。

張立業經理指出，MCTS-Project 2007是第一個在Project產品上的證照。接下來，在今年底之前還會有針對Project Server 2007的MCTS認證，以及針對Project Server 2007的MCITP認證及相關課程推出。
