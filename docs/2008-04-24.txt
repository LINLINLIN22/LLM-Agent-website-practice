標題: 微軟發表兩軟體更新版
原文: https://www.ectimes.org.tw/2008/04/%e5%be%ae%e8%bb%9f%e7%99%bc%e8%a1%a8%e5%85%a9%e8%bb%9f%e9%ab%94%e6%9b%b4%e6%96%b0%e7%89%88/
記者／黃逸珊

台灣微軟22日在台中發表最新研發的Windows Server 2008及Visual Studio 2008兩套軟體更新版，訴求以更人性化的管理平台，可透過虛擬化技術，大幅提升伺服器的作業效率，強化網路管理伺服器多項效能；微軟強調，企業投資1元改善伺服器管理系統，可帶進18元產值效能。

台灣中部地區地位逐漸提升，台灣微軟公司不敢小覷中部市場，首度將台中作為年度重量級產品發表的第一站，正式向企業界推出新一代作業Windows  Server2008及開發工具Visual Studio2008。

微軟營運暨行銷處總經理柯安力表示，這兩套新產品，歷經五年研發，目前全台約有85%以上的企業使用舊版產品，新產品推出後，預料將掀起一波升級潮，市場商機無限，對台灣企業提升全球競爭力，也將是一個新的里程碑。目前國內已有多家知名企業正在試用這兩套新產品，並且認為透過Windows Server 2008所提供的遠端存取程式功能，讓兩岸三地與全球化營運更有效率。並且這兩項新產品在虛擬化及核心技術上都有創新突破，讓企業員工在建構應用程式時更有效率，不必撰寫複雜程式，可以快速開發各個企業適用的軟體工具，並且滿足企業現在及未來十年的成長需求。

柯安力並表示，目前Windows Server的市佔率已超過七成，在發表會之後，預期市佔率會更加提高，但是為了回饋業界，Windows Server2008只維持原來的售價，並且依據企業的不同需求，提供不同的版本。
